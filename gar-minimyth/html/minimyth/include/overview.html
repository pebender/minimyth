MiniMyth is a small Linux distribution that turns a diskless computer into a MythTV frontend.

As of 2007-04-28, the latest version of MiniMyth contains:

  • MythTV 0.20.1 with fixes through SVN 13368, including the following plugins:
    • MythBrowser
    • MythDVD
    • MythGallery
    • MythGame (not integrated with MiniMyth configuration)
      • Atari 2600 emulation using Stella 2.3.5
      • Game Boy, Game Boy Color and Game Boy Advance emulation using VisualBoyAdvance 1.7.2 and Mednafen 0.8.1
      • Intellivision emulation using jzIntv 1.0-beta3
      • Nintendo Eterntainment System emulation using FCE Ultra 0.98.13-pre and Mednafen 0.8.1
      • Super Nintendo Eterntainment System emulation using ZSNES 1.51
    • MythMusic
    • MythNews
    • MythVideo
    • MythWeather
  • MythStream 0.17_2c (although it is labeled 0.17_2)
  • MPlayer 1.0rc1 and the openChrome patch
  • xine-ui 0.99.4 and xine-lib 1.1.6
  • Xorg 7.2
  • openChrome Xorg graphics driver SVN 318
  • NVIDIA Xorg graphics driver 1.0-9755
  • ALSA driver 1.0.14rc3
  • LIRC 0.8.1
  • LCDproc 0.5.1
  • Linux 2.6.17.14

In addition, there is a corresponding version of MiniMyth that contains MythTV 0.19 with fixes through SVN 13054.
