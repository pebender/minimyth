MiniMyth is a small Linux distribution that turns a diskless computer into a MythTV frontend.

As of 2006-12-06, the latest version of MiniMyth contains:

  • MythTV 0.20 with fixes through SVN 12172, including the following plugins:
    • MythDVD
    • MythGallery
    • MythMusic
    • MythNews
    • MythVideo
    • MythWeather
  • MythStream 0.17_2b (although it is labeled 0.17_2)
  • MPlayer CVS 20061202 and the openChrome patch
  • xine-ui 0.99.4 and xine-lib 1.1.3
  • Xorg 7.2-RC3
  • openChrome Xorg graphics driver SVN 247
  • NVIDIA Xorg graphics driver 1.0-9631
  • ALSA 1.0.13
  • LIRC 20061202Z
  • LCDproc 0.5.1
  • Linux 2.6.17.14

In addition, there is a corresponding version of MiniMyth that contains MythTV 0.19 with fixes through SVN 10505.
