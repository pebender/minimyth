MiniMyth variables
user_sleep_enter

This function is called just before entering sleep. It is called after the MiniMyth frontend is put into a known state and after X is stopped (if X was configured to restart on sleep).

If you are using your MiniMyth frontend to control other equipment (e.g. television), then you can use this function to turn off the equipment.

user_sleep_leave

This function is called just after leaving sleep. It is called before X is started (if X was configured to restart on sleep).

If you are using your MiniMyth frontend to control other equipment (e.g. television), then you can use this function to turn on the equipment.
