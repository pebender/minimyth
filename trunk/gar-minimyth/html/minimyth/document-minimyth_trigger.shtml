Sleep Trigger Point Functions
mm_trigger_sleep_enter

This function is called at the sleep enter trigger point, which occurs just before entering sleep. This trigger point is after the MiniMyth frontend is put into a known state and after X is stopped (if X was configured to restart on sleep).

If you are using your MiniMyth frontend to control other equipment (e.g. television), then you can use this trigger point to turn off the equipment.

mm_trigger_sleep_leave

This function is called at the sleep leave trigger point, which occurs just after leaving sleep. This trigger point is before X is started (if X was configured to restart on sleep).

If you are using your MiniMyth frontend to control other equipment (e.g. television), then you can use this trigger point to turn on the equipment.
