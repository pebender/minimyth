MiniMyth is a small Linux distribution that turns a diskless computer into a MythTV frontend.

The latest MiniMyth release contains:

  • MythTV 0.19 with SVN fixes through SVN 10505, including the following plugins:
    • MythDVD
    • MythGallery
    • MythMusic
    • MythNews
    • MythVideo
    • MythWeather
  • MythStream 0.l6_5
  • MPlayer 1.0pre8 with the openChrome patch
  • xine-ui 0.99.4 and xine-lib 1.1.1
  • Xorg 7.0
  • openChrome Xorg graphics driver SVN 208
  • NVIDIA Xorg graphics driver 1.0-8178
  • ALSA 1.0.12
  • LIRC 0.8.0
  • LCDproc 0.5.0
  • Linux 2.6.17.11

In addition, in the test download directory, you can find a test version of MiniMyth that includes MythTV 0.20 with SVN fixes through SVN 11163.
