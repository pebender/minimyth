MiniMyth Changelog

--------------------------------------------------------------------------------
This file contains the changes made to MiniMyth since the previous stable
MiniMyth release. For earlier changes see the yearly changelog files.

--------------------------------------------------------------------------------
Changes since 75 (2010-12-28):

Current MythTV versions
    MythTV trunk: version trunk.27420, trunk svn 27420.
    MythTV 0.24:  version 0.24.0,      release-0-24-fixes branch svn 27419.
    MythTV 0.23:  version 0.23.1,      release-0-23-fixes branch svn 27411.
    MythTV 0.22:  version 0.22.0,      release-0-22-fixes branch svn 27412.

Modified mediaplayers
    - Updated FFmpeg and MPlayer multi-threading patches.

Modified LIRC
    - Switched from LIRC bundled RC Linux drivers to in-tree Linux RC drivers.
        - The LIRC maintainer uses the in-tree Linux RC drivers rather than
          the LIRC bundled RC Linux drivers, resulting in LIRC bundled RC
          Linux drivers being less reliable than the in-tree Linux LIRC drivers.
        - The in-tree Linux RC drivers do not support the xbox receiver
          variant of the lirc_atiusb driver. However, as the LIRC lirc_atiusb
          driver no longer works with newer (2.6.36 and above) Linux versions,
          it was not supported anyway.

Improved Xorg
    - Modified udev rules so that event devices work correctly with Xorg 7.5.

Modified kernel
    - Added gspca kernel drivers (and included in the image when mythzoneminder
      is included in the image).
    - Added ati_remote and ati_remote2 drivers.
    - Removed the r8168 driver because the kernel 2.6.36.2 changelog claims
      fixes in the r8169 driver that address the issues previously addressed by
      using the r8168 driver.

Moved packages
    Moved xorg-7.5 to xorg-7.6.
